# Data

This directory documents the data used in the oil forecast evaluation project.

## Processed appendix tables

- `processed/benchmark_metrics.csv`: EIA versus martingale benchmark forecast-error metrics.
- `processed/bias_corrected_metrics.csv`: Recursive historical-bias correction results.
- `processed/rolling_model_metrics.csv`: Rolling AR and rolling AR plus futures model results.

## Full local project folder

The local project folder also contains processed forecast panels, raw EIA AEO Table 12 workbooks, FRED Brent and CPI downloads, World Bank supplementary forecast data, and Brent futures features. The most appendix-ready results are summarized in `outputs/project_summary.md`.
