# Raw EIA AEO Table 12 Archive

The local project folder contains AEO Brent price forecast workbooks for vintages 2013-2023, 2025, and 2026. AEO 2024 is excluded because that vintage was not released.

These files are parsed by the pipeline into the EIA Brent forecast panel used for the 3-year and 5-year forecast evaluation.

Source pages:

- https://www.eia.gov/outlooks/aeo/tables_ref.php
- https://www.eia.gov/outlooks/archive/
