# 长期油价预测评估项目报告草稿

## 1. 项目目标

本项目围绕课程 proposal 中提出的核心问题展开：**长期油价预测是否足够可靠，能否为上游油气企业的资本开支和项目立项提供有用信息**。具体而言，我们关注：

- 机构长期油价预测是否存在系统性偏差
- 机构预测是否优于简单的 naive benchmark
- 引入时间序列信息和市场信息后，是否能够改进预测表现

考虑到 upstream oil & gas 项目的投资周期通常较长，本项目将“长期预测”定义为：

- `3-year ahead forecast`
- `5-year ahead forecast`

研究对象定义为：

- **Brent Spot Price**
- 单位统一为 **real 2025 USD per barrel**

## 2. 数据来源

本项目当前版本主要使用以下公开数据源：

### 2.1 EIA AEO forecast vintages

主分析使用 U.S. Energy Information Administration (`EIA`) 的 `Annual Energy Outlook (AEO)` Table 12 中的 Brent price forecast。我们从 2013 到 2026 的公开 AEO 页面中提取各 vintage 的 Brent 路径。

说明：

- `2024` 年没有对应 AEO release，因此没有 2024 vintage
- 当前可用于 realized evaluation 的主样本共覆盖 `11` 个 EIA publication years

### 2.2 Realized Brent price

真实 Brent 年均价来自 FRED 的：

- `DCOILBRENTEU`

先由日频数据聚合为 annual average，再统一转换为 `real 2025 dollars`。

### 2.3 CPI deflator

价格平减使用 FRED 的：

- `CPIAUCSL`

其作用是将不同 base year 下的价格统一转换到 `2025 constant dollars`，保证不同 EIA vintage 和 realized price 之间可比。

### 2.4 Futures proxy

在 improvement model 中，我们使用 Yahoo Finance 的：

- `BZ=F`

作为 Brent front-month futures 的公开 proxy，用于构造 `AR + Futures` 模型中的市场信息变量。

### 2.5 Supplementary source

另外，项目也下载了 World Bank 的历史 commodity price forecast archive。  
但需要说明的是，World Bank 公开表中的油价对象是 `Crude oil, avg`，并非严格意义上的 Brent，因此当前版本将其视为 supplementary material，而不是主结果的一部分。

## 3. 方法设计

## 3.1 Forecast evaluation

对每个 EIA vintage 和每个预测 horizon，我们构造 forecast error：

`forecast error = forecast - realized`

并计算以下评价指标：

- `ME (Mean Error)`：用于识别系统性 optimism / pessimism bias
- `MAE (Mean Absolute Error)`
- `RMSE (Root Mean Squared Error)`
- `MAPE (Mean Absolute Percentage Error)`

其中：

- `ME > 0` 表示平均上偏，即 forecast 偏乐观
- `ME < 0` 表示平均下偏，即 forecast 偏保守

## 3.2 Benchmark

为了判断机构预测是否具有增量价值，我们设置了一个 `random walk benchmark`：

- benchmark forecast 取 forecast 发布时点之前最近一个完整年度的实际 Brent 年均价

这个 benchmark 对应 proposal 中的 naive expectation，对长期 forecast evaluation 是一个合理的基准。

## 3.3 Improvement models

在 benchmark evaluation 之外，我们还实现了两个 rolling model：

### (1) Rolling AR

使用历史 annual Brent price 的滞后项作为特征，包括：

- `lag1`
- `lag2`
- `lag3`

通过 expanding-window 的方式做 out-of-sample prediction。

### (2) Rolling AR + Futures

在 AR 的基础上加入市场信息 proxy，包括：

- release month 的 futures price
- futures 与最近实际年均价之间的 basis

该模型试图对应 proposal 中“rolling autoregressive model augmented with oil futures information”的思路。

## 4. 主要结果

当前主结果如下。

### 4.1 EIA vs Random Walk

`3-year ahead`

- EIA: `ME = 20.44`, `MAE = 26.01`, `RMSE = 34.97`, `MAPE = 0.388`
- Random Walk: `ME = 17.25`, `MAE = 31.66`, `RMSE = 42.99`, `MAPE = 0.438`

`5-year ahead`

- EIA: `ME = 26.64`, `MAE = 28.09`, `RMSE = 34.32`, `MAPE = 0.381`
- Random Walk: `ME = 13.79`, `MAE = 38.90`, `RMSE = 47.34`, `MAPE = 0.512`

### 4.2 对结果的解释

从当前样本看，EIA 的长期 Brent forecast 在 `3-year` 和 `5-year` horizon 上都优于 random walk benchmark，尤其体现在 `RMSE` 和 `MAE` 上。这说明：

- EIA forecast 并非完全没有信息量
- 相比于简单地延续最近一年油价，EIA 的长期预测至少在误差控制上更好

但与此同时，EIA 的 `ME` 在两个 horizon 上都显著为正：

- `3-year ME = 20.44`
- `5-year ME = 26.64`

这说明在当前样本中，EIA forecast 存在明显的**正向偏差**，也就是平均上偏，倾向于高估未来 Brent price。  
如果把研究问题放回 upstream capital planning 的决策背景下，这个结论是重要的，因为系统性高估油价可能导致：

- 对 future cash flow 过度乐观
- NPV / IRR 被高估
- 项目立项时点偏早
- 高油价情景下的 overinvestment 风险上升

### 4.3 Rolling model results

`Rolling AR`

- 3-year: `RMSE = 48.95`
- 5-year: `RMSE = 51.24`

`Rolling AR + Futures`

- 3-year: `RMSE = 52.51`
- 5-year: `RMSE = 53.23`

在当前实现下，这两个 improvement model 并没有超过 EIA 主预测结果。换句话说：

- 简单 rolling AR 没有战胜 EIA
- 加入公开 futures proxy 后，模型也没有显著改善

这说明至少在本版数据和建模设定下，EIA 自身的长期 forecast 仍然优于我们当前的 econometric baseline。

## 5. 结果总结

本项目当前可以支持以下几个初步结论：

1. `EIA` 的长期 Brent forecast 在 `3-year` 和 `5-year` horizon 上整体优于 random walk benchmark。
2. 尽管如此，`EIA` forecast 存在较明显的 positive bias，即平均上偏。
3. 从资本预算角度看，EIA 预测可以作为有信息量的输入，但不应直接无调整地用于 DCF 模型。
4. 一个合理的管理含义是：企业在使用机构长期 forecast 时，应考虑加入 bias adjustment 或 conservative haircut。
5. 当前版本的 rolling AR 和 AR+Futures 模型还没有显示出超越 EIA 的证据。

## 6. 局限性

当前版本仍有几个需要明确说明的局限。

### 6.1 IEA 未纳入主分析

proposal 中原本设想比较多个机构，包括：

- `EIA`
- `IEA`
- `World Bank`

但在当前项目时间内，我们只把 `EIA` 完整纳入主分析。  
`World Bank` 只作为补充，因为其 forecast object 是 `Crude oil, avg` 而不是严格 Brent。  
`IEA` 历史长周期 forecast 的 machine-readable public archive 没有在本版中整合。

### 6.2 Futures 信息仍是 proxy

proposal 设想的是 oil futures information。当前使用的 `BZ=F` 是公开可获得的 front-month proxy，而不是完整 long-dated Brent futures curve，因此市场信息模块仍然比较粗糙。

### 6.3 Sample size 仍有限

由于长期预测评估依赖 vintage structure，而且必须等待 realized value 出现，`3-year` 和 `5-year` horizon 的有效样本天然有限。因此：

- 结果有方向性意义
- 但统计稳定性仍受样本量约束

## 7. 下一步建议

如果继续推进这个项目，建议优先做以下几项扩展：

1. 补充 `IEA` 历史 vintages，完成真正的 cross-institution comparison
2. 将 World Bank 的 `Crude oil, avg` 系列单独做 robustness check
3. 引入更完整的 Brent futures term structure，而不是仅使用 front-month proxy
4. 在 improvement analysis 中加入更明确的 bias-correction framework
5. 在最终展示中增加：
   - forecast vintages vs realized path 的图
   - 不同 horizon 的 error decomposition
   - bias 对 DCF / NPV 的经济含义讨论

## 8. 结论

总体来看，本项目当前版本已经完成了 proposal 中最核心的一部分：  
**对 EIA 长期 Brent forecast 的 3 年和 5 年 horizon 进行系统评估，并判断其是否优于 naive benchmark，同时检验其是否存在系统性偏差。**

目前最重要的结论是：

- `EIA` 的长期 forecast 有信息量，优于 random walk
- 但 `EIA` forecast 在样本中表现出明显 optimism bias

因此，从企业决策角度，EIA forecast 可以使用，但更合适的做法不是直接照搬，而是进行适当的 bias adjustment，再用于 upstream investment appraisal。
