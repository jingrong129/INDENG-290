# Outputs

The local project folder contains generated appendix materials:

- `project_summary.md`: compact summary of key tables and findings.
- `analysis_outputs.xlsx`: consolidated workbook exported by the pipeline.
- `benchmark_metrics.png`: EIA versus martingale benchmark chart.
- `eia_vintages_vs_actual.png`: forecast vintages versus realized Brent.
- `bias_correction_metrics.png`: recursive bias-correction chart.
- `cpi_brent_relationship.png`: CPI and Brent diagnostic chart.
- `rolling_model_rmse.png`: rolling-model comparison chart.
- `final_project_report.pdf`: rendered final report.
- `oil_forecast_evaluation_presentation.pptx`: project presentation.

For the GitHub appendix, `project_summary.md` and the processed CSV files provide the most direct evidence tables.
